# Simple Management System

## Overview
This is a **beginner-friendly Management System** built using **Python**, **Tkinter** for the graphical user interface (GUI), and **SQLite3** as the database.  

The goal of this project was to practice **Python programming, GUI development, and database integration** while building a functional desktop application.

---

## Features

- Add, view, update, and delete records  
- User-friendly interface with **Tkinter GUI**  
- Data stored persistently in **SQLite3 database**  
- Simple and intuitive navigation for beginners  

---

## Tech Stack

- **Programming Language:** Python  
- **GUI Library:** Tkinter  
- **Database:** SQLite3  

---

## Goal of the Project

The main goals of this project were:

1. Learn the basics of **Python GUI development**  
2. Practice working with **databases** using SQLite3  
3. Implement **CRUD operations** (Create, Read, Update, Delete)  
4. Build a **complete, functional desktop application**  
5. Improve understanding of **Python programming structure**  

---

## What I Learned

During this project, I learned:

- How to use **Tkinter** to create windows, buttons, labels, and input fields  
- How to **connect Python to SQLite3** and perform database operations  
- How to implement **CRUD functionality** in a Python application  
- How to structure a beginner Python project for better readability  
- Basic **error handling** and input validation
